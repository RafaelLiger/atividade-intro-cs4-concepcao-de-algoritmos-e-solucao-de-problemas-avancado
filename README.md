## Exercício 1

Este exercício foi desenvolvido para praticar vetores, estruturas condicionais e laços de repetição em Portugol. O objetivo é ordenar os números do vetor em ordem decrescente, trabalhando comparação e troca de valores.

---

## Exercício 2

Este exercício tem como objetivo praticar leitura de dados em vetores, identificação de números pares e índices ímpares, além do cálculo de soma e média. Ele ajuda no desenvolvimento da lógica de programação básica em Portugol.
